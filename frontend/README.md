# Frontend (Next.js)

OddFinds frontend application built with Next.js 15 App Router and Tailwind v4.

## Run
- `npm run dev`

## Key Areas
- `app/` route groups for shop, auth, user, admin
- `components/` shared UI and layout primitives
- `stores/` Zustand client stores (cart, wishlist, UI, admin)
- `lib/` integrations (auth, payments, AI, email, shipping, seo)
- `app/api/` route handlers (commerce, admin, AI, webhooks)
